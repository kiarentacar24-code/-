## ✅ 완성된 기능

### 1. 가독성 개선 ✨
- ✅ 배경색과 텍스트 색상 최적화 (잘 보이도록)
- ✅ 카드형 레이아웃으로 내용이 잘리지 않음
- ✅ 외장색상/내장색상 모두 표시
- ✅ 롯데렌터카 로고 배경 통일 및 크기 확대

### 2. 재고 표시 📊
- ✅ **롯데**: 판매가능재고 / 입고재고 / 정상재고 3종 표시
- ✅ **현대/기아**: 정상재고만 표시

### 3. 차종 필터 🔍
- ✅ 차종명 한글 표시 (예: AVANTE → 아반떼)
- ✅ 롯데/현대/기아 차종 중복 제거 (1개로 표시)
- ✅ 제조사별 필터링
- ✅ 검색 기능

### 4. 텍스트 복사 📋
- ✅ 포맷 1: `카메이커, 차종코드, 옵션코드, 외장코드, 내장코드, 출하장`
- ✅ 포맷 2: `차종: [차종명] | 옵션: [옵션명] | 외장색상/내장색상: [색상]`

### 5. 엑셀 기반 자동 로드 📂
- ✅ 브라우저에서 SheetJS로 엑셀 파일 직접 로드
- ✅ 3개 파일 (롯데/현대/기아) 자동 통합
- ✅ 매일 교체만 하면 자동 반영

## 🔧 커스터마이징

### 차종명 한글 추가
`index.html` 파일에서 `MODEL_NAME_MAP` 검색:

```javascript
const MODEL_NAME_MAP = {
  'AVANTE': '아반떼',
  'SONATA': '쏘나타',
  // 새 차종 추가
  'NEW_MODEL': '새차종',
};
```

### 로고 변경
`index.html` 파일에서 `logoImg` 검색:

```html
<img class="logoImg" src="새로운URL" alt="롯데렌터카 로고" />
```

### 색상 테마 변경
`index.html` 파일에서 `:root{` 검색:

```css
:root{
  --accent:#e60012;      /* 메인 색상 */
  --accent2:#2563eb;     /* 보조 색상 */
}
```

## 🔄 매일 엑셀 업데이트

### 방법 1: GitHub 웹사이트 (30초)

```bash
git commit -m "Update inventory"
git push
```

## ❓ 자주 묻는 질문

### Q1: 데이터가 안 보여요
**A**: 브라우저 F12 → Console 탭에서 에러 확인
- 파일명이 정확한가요? (`lotte.xlsx`, `hyundai.xls`, `kia.xls`)
- 파일이 `data/` 폴더 안에 있나요?
- 엑셀 컬럼명이 샘플과 일치하나요?

### Q2: 차종명이 영어로 나와요
**A**: `index.html`의 `MODEL_NAME_MAP`에 매핑 추가

### Q3: 업데이트가 반영 안돼요
**A**: 브라우저 강제 새로고침 (Ctrl+Shift+R / Cmd+Shift+R)

### Q4: 404 에러가 나요
**A**: Settings → Pages에서 Source가 `main` 브랜치로 설정되었는지 확인
